# DBMS Assignment 2

Database management system project with React frontend and Express backend.

## Structure

```
client/     React application
server/     Express API server
db/         SQL scripts
```

## Setup

**Server**
```bash
cd server
npm install
npm start
```

**Client**
```bash
cd client
npm install
npm start
```

## Database

Run SQL scripts in `db/` folder in numerical order.

## Stack

- Frontend: React, TailwindCSS
- Backend: Node.js, Express, MySQL2
- Database: MySQL
